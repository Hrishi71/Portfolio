# Modern Portfolio Website

A sleek, modern portfolio website built with React and Vite, featuring a dark theme, smooth animations, and responsive design.

## Features

- **Dark Theme**: Modern dark color scheme with glassmorphism effects
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: CSS transitions and keyframe animations
- **Typing Effect**: Animated typing effect in the hero section
- **Interactive Elements**: Hover effects and micro-interactions
- **Modern Typography**: Clean, readable fonts with gradient text effects
- **Sections**: Home, About, Skills, Projects, Contact
- **Social Links**: Footer with social media links

## Technologies Used

- React 18
- Vite
- CSS3 with modern features (backdrop-filter, CSS variables)
- Google Fonts (Inter)

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customization

- Update personal information in `src/App.jsx`
- Modify colors and styles in `src/App.css`
- Add your own projects and skills
- Replace placeholder content with your actual portfolio items

## Build

To build for production:
```
npm run build
```

## Deployment

The built files in the `dist` folder can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.
